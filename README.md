## Getting started


### Take a git clone or fork

### Running the project

However you get the code, you can install dependencies and run the project in development mode with:

```bash
cd fedi-todo
npm install # or yarn
npm run dev
```

Open up [localhost:3000](http://localhost:3000) and start clicking around.
